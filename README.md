## Installation
### Download latest releases
For releases, click [here](https://github.com/Ashcaberze/Harley/releases). 

### Clone the repo & run
```bash
  cd "Program Files"
  git clone git@github.com/spirizeon:harley.git
  cd "Harley"
  pip install -r requirements.txt
  ./harley_unopt.py
```
